# (void) get(field, options = {})

Override Model#get to access associations as well as fields.

Examples:

Retrieve a Backbone model property by delegating to Backbone.Model#get 

(new Task(title: 'Sweep the garage')).get('title') # => 'Sweep the garage'

Retrieve a Brainstem Association. The returned association may be a collection or a model. 

task = new Task(id: 1)
task.fetch({include: 'assignees'})
  .done( -> task.get('assignees')) # => Assignees Collection

Parameters:

  • field ( String ) — Name of the property the model is attempting to access.
  • options ( Object ) — Configuration object

Options Hash: (options):

  • Boolean ( linked ) — When set to 'true', a collection singleton is created for this creation. The model will listen to the collection for change events and update its (the model) own internal records of association Id's. Future calls to #get for this association will always return the same instance of the collection.

# (Boolean) associationsAreLoaded(associations)

Determines if all of the provided associations have been loaded for this model. If no associations are provided, all associations are assumed.

Examples:

Check if all associated models are loaded for a particular association 

model.associationsAreLoaded(["project", "task"]) # => true|false

Check if all associated models are loaded for all of this models associations 

model.associationsAreLoaded() # => true|false

Parameters:

  • associations ( Array ) — Optional array of strings detailing the associations to check

Returns:

  • Boolean
