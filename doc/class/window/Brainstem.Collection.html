Class Method Details

. (void) getComparatorWithIdFailover(order)

Returns a collection comparator using the provided options. If the two sorting values are equal, the comparator will default to a model's id.

Examples:

Sort by property {duration} in descending order 

Collections.Tasks.getComparatorWithIdFailover('duration:desc') # sorts descending duration

Parameters:

  • order ( String ) — Attribute to use for ordering. Syntax of order is a {property}:{direction}

. (void) getComparator(field)

Returns a comparator function. Assumes the field value on the model's attributes is a number.

. (void) pickFetchOptions(options)

Picks only options related to a fetch operation out of the passed object

Constructor Details

# (void) constructor(models, options)

Constructor delegates to Backbone#Collection. The collection has an internal flag, {setLoaded} to denote whether or not the collection has been fetched from the server yet.

Parameters:

  • models ( Array ) — An array of Brainstem Models that the collection will contain. These models will be set on the collection by Backbone.
  • options ( Object ) — An options object. A number of these properties are used as fetch options.

# (void) update(models)

Manually update the collection with an array of new models. For each model, if the collection does not contain the model it will be #added. If the model already exists in the collection it will be #set with the new model's attributes.

Parameters:

  • models ( Array ) — Array of models to update the collection with.

# (void) reload(options)

Reset the collection, removing all models contained within the collection. The collection then fetches itself with its lastFetchOptions.
